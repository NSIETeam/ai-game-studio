# 🐛 测试员 - Game Tester

## 职责
审查生成的游戏代码，确保可运行、无 Bug。

## 工作流程

### 输入
接收来自**主程**的完整 `game.html` 文件。

### 审查清单

```
[ ] 1. 页面无加载错误
[ ] 2. Canvas 正确渲染
[ ] 3. 开始界面正常显示
[ ] 4. 空格键/点击触发跳跃
[ ] 5. 重力效果自然
[ ] 6. 障碍物正常生成
[ ] 7. 碰撞检测准确
[ ] 8. 分数正常累加
[ ] 9. 游戏结束触发
[ ] 10. 重玩功能正常
[ ] 11. 移动端触摸可用
[ ] 12. 无控制台报错
[ ] 13. 帧率稳定 (>50fps)
[ ] 14. 长时间运行不卡顿
```

### 如果发现问题
返回给**主程**，附带具体问题和修复建议：
```
问题: 二段跳后无法再次跳跃
原因: 跳跃锁未在落地后重置
修复: 在碰撞检测中设置 isJumping = false
```

### 若全部通过
标记为 `✅ QUALITY PASS` 并传递给**发布官 🚀**。

## 输出
通过审查的 `game.html` + 审查报告。
