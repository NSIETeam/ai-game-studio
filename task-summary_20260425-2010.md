# AI Game Studio — Velorah 风格重构

## 目标
将 AI Game Studio 展示网站从原有的赛博/游戏风格全面重构为 D:\project\website_prompt 中描述的 **Velorah 玻璃拟态风格**（深色背景 + liquid-glass 效果 + Instrument Serif 字体 + 极简排版）。

## 技术方案
- 原 prompt 使用全屏视频做视觉深度 → 替换为 **Canvas 粒子（星点）+ 星云柔光**（`src/main.js` 中实时渲染）
- 保留所有 6 个 Agent 流水线 + 作品展示 + 能力 + 技术栈 + CTA 内容
- 去除所有网格线、边框、卡片阴影 → 替换为 **liquid-glass 拟态**（渐变描边层 `::before`）
- 导航固定在顶部，滚动时出现 blur 背景

## 修改的文件
| 文件 | 变化 |
|------|------|
| `index.html` | 替换字体为 Instrument Serif + Inter，改用英文 |
| `src/main.js` | 注入 Canvas 粒子背景（3层星点 + 星云渐变） |
| `src/style.css` | 完全重写：liquid-glass 效果、新动画、深色主题 |
| `src/App.vue` | 完全重写：新 Hero/导航/流水线卡片/作品/能力/技术栈/CTA/Footer |

## 设计系统
- **背景**: hsl(201, 100%, 13%) 深海军蓝 + Canvas 粒子星空
- **字体**: Instrument Serif (标题/Logo) + Inter (正文)
- **玻璃效果**: `::before` 渐变描边边框，`backdrop-filter: blur(4px)`
- **交互**: hover scale(1.03) 弹性感，滚动触发 reveal 动画
- **色彩**: 紫(252,56%,65%) + 粉(342,92%,66%) 渐变为 accent
