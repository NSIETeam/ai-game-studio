# AI Finance Legion — Pipeline Workflow

```
用户输入财务数据 -------------------------------------------+
                                                             |
                                                             v
   ┌──────────────────────────────────────────────────────────┐
   │  Step 1: 财务审计官 (Finance Auditor)                  │
   │  数据清洗 → 分类归账 → 异常标记                        │
   └──────────────────┬───────────────────────────────────────┘
                      | 分类账 + 异常标记
                      v
   ┌──────────────────────────────────────────────────────────┐
   │  Step 2: 财报分析官 (Financial Analyst)                 │
   │  四大报表 → 关键比率 → 趋势分析                        │
   └──────────────────┬───────────────────────────────────────┘
                      | 报表 + 比率 + 趋势
                      v
   ┌─────────────────┐   ┌──────────────────────┐
   │ Step 3a: 投资参谋  │   │  Step 3b: 税务合规官  │  ← 并行
   │ 投资建议 + 配比   │   │  税额计算 + 发票合规  │
   └────────┬─────────┘   └──────────┬─────────────┘
            | 投资建议                | 税务报告
            v                         v
   ┌──────────────────────────────────────────────────────────┐
   │  Step 4: 风控审计官 (Risk Controller)                   │
   │  反欺诈 → 合规审计 → 信用评估 → 压力测试               │
   │  接收 Step 1-3 全部数据进行交叉验证                     │
   └──────────────────┬───────────────────────────────────────┘
                      | 风险报告 + 审计报告
                      v
   ┌──────────────────────────────────────────────────────────┐
   │  Step 5: 首席财务官 (CFO)                               │
   │  全局汇总 → 战略建议 → 最终报告                         │
   └──────────────────┬───────────────────────────────────────┘
                      |
                      v
               最终财务报告 🎯
```

## 并行执行
- Step 3a（投资参谋）和 Step 3b（税务合规官）可并行运行
- 风控官接收 Step 1~3 的全部输出进行交叉验证

## 执行方式（OpenClaw Subagent）

```
1. spawn subagent: "你作为财务审计官..." → 产出分类账
2. spawn subagent: "你作为财报分析官，基于以下数据..." → 产出报表
3a. spawn subagent: "你作为投资参谋，基于..." → 产出投资建议
3b. spawn subagent: "你作为税务合规官，基于..." → 产出税务报告
4. spawn subagent: "你作为风控审计官，综合所有数据..." → 产出风险报告
5. spawn subagent: "你作为CFO，汇总所有报告..." → 产出最终报告
```

## 真实 API 集成
- **neodata-financial-search**: 股票/基金/指数/宏观实时数据
- **multi-search-engine**: 行业费用标准、研报摘要、政策更新
